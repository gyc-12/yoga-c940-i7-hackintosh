## 安装条件
参见网络上的黑苹果安装教程，在此不做赘述。

### 注意
1、请更换原装固态PM981a，在该固态下就算强行使用恢复版镜像，也会造成超过4G大小的文件读写出错。

2、解锁CFG和设置DVMT到64m。可以通过刷advanced版bios来设置（不建议，比较复杂而且容易出错，建议使用提供的文件）。

修改bios有风险，请提前备份好bios，以免开不了机。

3、设备信息已经删除，请自行在config.plist里添加设备信息。

4、测试运行在 macos14.4.1
